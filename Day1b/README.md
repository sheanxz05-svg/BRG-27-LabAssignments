Today I learned more about Linux system.

## Activities completed

- Listed Linux services using systemctl
- Checked the status of the cron service
- Viewed file permissions using ls -l

- Searched for files using find

## What I learned

I learned that Linux services run in the background and are managed using systemctl.
I also learned how file permissions work and how chmod changes access rights. 
The find and grep commands are useful for locating files and searching for information quickly.
