# Day 2b

## Cloud Web Server Deployment

### Objective

The objective of this lab was to bring out a 
cloud-based Ubuntu server using Amazon EC2 and install Apache web server.

### AWS EC2 Setup

- Created AWS account
- Launched Ubuntu EC2 instance
- Configured security group with SSH (22) and HTTP (80)
- Connected to EC2 and installed Apache

### Apache Testing

I verified the installation by opening the
public IP address in a browser and viewing the Apache default page.

### Bash Coding

I created and executed several Bash scripts:

- hello_world.sh
- system_info.sh
- resource_monitor.sh

These scripts helped me understand Bash syntax, loops, and system monitoring.

### What I Learned

This activity helped me understand how cloud servers are created and managed. I also learned how Bash scripts can automate tasks and improve efficiency.
